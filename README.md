# StudySync AI

StudySync AI is an intelligent machine learning platform that helps students improve their academic performance through personalized insights.

## Features (Planned)

- Student Performance Prediction
- Personalized Study Planner
- Weak Subject Detection
- AI-powered Recommendations
- Learning Analytics Dashboard

## Tech Stack

- Python
- Flask
- Scikit-Learn
- XGBoost
- CatBoost
- Pandas
- NumPy
